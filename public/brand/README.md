# Brand — Cesar's Restaurant Appliances Repair

Flat vector logo system. Refined **CR** monogram + clean industrial wordmark.
Direction: premium commercial-equipment service (Tesla / Rivian / Caterpillar /
Sub-Zero / Wolf), not a contractor/handyman mark.

## Files

| File | Use |
|------|-----|
| `logo-horizontal.svg` | **Primary** horizontal logo — transparent background, dark text (for light/neutral surfaces) |
| `logo-horizontal-white-bg.svg` | White background version |
| `logo-horizontal-dark-bg.svg` | Dark background version (light text, brighter red) |
| `logo-icon.svg` | Icon-only — the CR monogram badge |

All are SVG (scalable vector). Flat — no gradients, no 3D, no shadows.

## Colors

| Role | Hex |
|------|-----|
| Black | `#0a0b0d` |
| Charcoal (badge) | `#16181c` / `#1a1d21` |
| Steel gray | `#6b7280` |
| Light steel (mono "C", subline on dark) | `#e7ebf0` / `#9aa3af` |
| Deep red accent (mono "R", apostrophe) | `#e11d2a` (brighter `#ff3b4a` on dark) |
| Wordmark on light | `#0e0f12` |

## Typography

Wordmark set in a heavy geometric/grotesque sans (delivered with a
`Helvetica Neue / Arial` stack for universal rendering; the website renders the
same mark in its display face, **Sora**). "CESAR'S" is uppercase, bold, tight;
"RESTAURANT APPLIANCES REPAIR" is small, letter-spaced steel-gray.

> For print or third-party handoff, convert the `<text>` elements to outlines so
> rendering doesn't depend on the viewer's installed fonts.

## Clear space & minimums

- Keep clear space around the logo equal to the height of the "C" in the badge.
- Minimum width: ~150px (horizontal) / 24px (icon) for legibility.
